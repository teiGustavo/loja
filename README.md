# ShopGestor - Sistema de Gerenciamento de Loja Virtual

### Site / Domínio: [ShopGestor.site](http://shopgestor.site)

## Considerações:
- Desenvolvido por: Gustavo Teixeira
- Linguagens: HTML5, CSS3, JS, PHP, SQL
